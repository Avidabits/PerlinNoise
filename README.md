Descripción de la obra:
El ruido de Perlin es una función matemática que utiliza interpolación entre un gran número de gradientes precalculados de vectores que construyen un valor que varía pseudo-aleatoriamente en el espacio o tiempo. 
Este tipo de ruido es frecuentemente utilizado en imágenes generadas por ordenador para simular variabilidad en todo tipo de fenómenos sin perder continuidad, acercándose así a un aspecto más natural.

En este obra, de exploración puramente técnica, realizo experimentos con Processing y el ruido de Perlin para realizar trabajos serigráficos.

Pinchando con el ratón en un punto de la pantalla se inicia un patrón gobernado por ruido de Perlin
